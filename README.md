## 使用说明

1. 将 `setting.json` 文件中的 token 改为自己的 Discord token
2. 运行 Python 脚本
3. 按照终端提示选择操作

## 功能选项

- `A`: 下载所有服务器的表情和贴纸
- `数字`: 选择特定服务器 (如: 1)
- `R`: 刷新服务器列表
- `Q`: 退出程序
